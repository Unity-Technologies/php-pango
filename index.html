php-pango by mgdm

Pango wrapper for PHP

This is a wrapper for the Pango text layout library for PHP.

Dependencies

* PHP 5.2+ * Pango 1.12+

Install

It is available using the PECL installer: 

pecl channel-discover pecl.mgdm.net
pecl install mgdm/Pango-beta
